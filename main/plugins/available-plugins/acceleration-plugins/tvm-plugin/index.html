TVM PluginðŸ”—

The TVM plugin for vAccel implements image inference vAccel operations with TVM.

Supported operationsðŸ”—

  • Image classification

Installing TVM C/C++ API filesðŸ”—

You can find instructions on how to install the required files here. The rest of this guide assumes TVM is installed at /opt/tvm and its' libraries are at /opt/tvm/build.

Installing the pluginðŸ”—

You can get the latest TVM plugin binary release from the Releases page of the vAccel repository. Releases include binaries for x86_64/aarch64 Ubuntu-based systems.

TARðŸ”—

To install the TAR binary package of the latest TVM plugin release:

wget https://github.com/nubificus/vaccel/releases/download/v0.7.0/vaccel-tvm_0.2.0_amd64.tar.gz
# Replace '/usr/local' below with the desired installation prefix
tar xfv vaccel-tvm_0.2.0_amd64.tar.gz --strip-components=2 -C /usr/local

wget https://github.com/nubificus/vaccel/releases/download/v0.7.0/vaccel-tvm_0.2.0_arm64.tar.gz
# Replace '/usr/local' below with the desired installation prefix
tar xfv vaccel-tvm_0.2.0_arm64.tar.gz --strip-components=2 -C /usr/local

Latest artifactsðŸ”—

You can also find prebuilt artifacts of the latest TVM plugin revision at:

# TAR
https://s3.nbfc.io/nbfc-assets/github/vaccel/plugins/tvm/rev/main/x86_64/release/vaccel-tvm-latest-bin.tar.gz

# TAR
https://s3.nbfc.io/nbfc-assets/github/vaccel/plugins/tvm/rev/main/aarch64/release/vaccel-tvm-latest-bin.tar.gz

UsageðŸ”—

To specify TVM plugin as the selected plugin for vAccel execution:

export VACCEL_PLUGINS=libvaccel-tvm.so

Ensure TVM, vAccel and the TVM plugin libraries are in the library search paths before trying to use the plugin.

Running an exampleðŸ”—

Export the necessary variables:

export LD_LIBRARY_PATH=/opt/tvm/build
export VACCEL_PLUGINS=libvaccel-tvm.so
# Optionally, for verbose output
export VACCEL_LOG_LEVEL=4

Assuming vAccel is installed at /usr/local, you can run an image classification with a ResNet model from https://s3.nbfc.io/models/tvm/x86_64/resnet18-v2-7.so with:
